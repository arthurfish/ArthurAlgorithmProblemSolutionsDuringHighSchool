# 信封问题

## 题目描述

某人写了 $n$ 封信和 $n$ 个信封，如果所有的信都装错了信封。求所有信都装错信封共有多少种不同情况。

## 输入格式

一个信封数 $n$，保证 $n \le 20$。

## 输出格式

一个整数，代表有多少种情况。

## 样例 #1

### 样例输入 #1

```
2
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
3
```

### 样例输出 #2

```
2
```

## 提示

对于 $100 \%$ 的数据，$1 \le n \le 20$。

# GCX的AC代码
```cpp
#include<iostream>
using namespace std;

int main(){
    int N = 0;
    cin >> N;
    unsigned long long *d = new unsigned long long[1024];
    d[1] = 0;
    d[2] = 1;
    if(N == 1){
	cout << 0 << endl;
	return 0;
    }else if(N == 2){
	cout << 1 << endl;
	return 0;
    }
    for(int i = 3; i <= N; i++)
	d[i] = (i - 1) * (d[i - 1] + d[i - 2]);
    cout << d[N] << endl;
}

```
